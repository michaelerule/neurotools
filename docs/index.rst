Welcome to Neurotools's documentation!
======================================

.. toctree::
   :maxdepth: 3
   :caption: Contents:

This is a "catch-all" project for computational and
statistical neuroscience Python routines. Code is poorly tested and
poorly documented. Certain individual modules are somewhat more 
stable and useful. Please refer to specific modules 
(see Module Index) for further documentation.
